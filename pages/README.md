# Pages

For clean, SEO-friendly URLs (e.g. `/services.html`, `/about.html`) the page files in
this template live in the **project root**, not inside this folder. Each is a thin
shell whose content is rendered from `config.json` by `js/pages.js`.

This `/pages` folder is part of the conventional structure and is reserved if you
prefer to relocate secondary pages here later. The auto-generated local-SEO city pages
live in `/locations`.
